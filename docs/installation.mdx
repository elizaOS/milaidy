This page covers the supported ways to install Milady and verify a working setup.

## Requirements

- Node.js `>=22` (required for runtime/tooling)
- Terminal access (or PowerShell on Windows)
- Internet access for installation and cloud providers

## Option A: Install scripts (recommended)

### macOS / Linux / WSL

```bash
curl -fsSL https://milady-ai.github.io/milady/install.sh | bash
milady setup
```

### Windows (PowerShell)

```powershell
irm https://milady-ai.github.io/milady/install.ps1 | iex
milady setup
```

## Option B: npm global install

```bash
npm install -g miladyai
milady setup
```

## Option C: Desktop release builds

Download the latest release artifacts from GitHub Releases (macOS, Windows, Linux).

After installing, still run onboarding/setup from the app/CLI as prompted.

## Verify install

Run:

```bash
milady --version
milady --help
```

If command is not found, restart your shell and verify your PATH.

## First run

```bash
milady
```

On first run, complete onboarding (name, style, provider, optional wallet).

## Security baseline

Milady binds to loopback by default. If exposing to network interfaces, set an API token first:

```bash
echo "MILADY_API_TOKEN=$(openssl rand -hex 32)" >> .env
```
