# Configuration

## Config File

Lives at `~/.milaidy/milaidy.json`

```json5
{
  agent: {
    name: "mila",
    model: "anthropic/claude-opus-4-5",
  },
  env: {
    ANTHROPIC_API_KEY: "sk-ant-...",
  },
}
```

Or use `~/.milaidy/.env` for secrets.

## Ports

| Service | Default | Env Override |
|---------|---------|--------------|
| Gateway (API + WebSocket) | `18789` | `MILAIDY_GATEWAY_PORT` |
| Dashboard (Web UI) | `2138` | `MILAIDY_PORT` |

```bash
# custom ports
MILAIDY_GATEWAY_PORT=19000 MILAIDY_PORT=3000 milaidy start
```

## Model Providers

Milaidy supports multiple AI model providers. Set the corresponding environment variable to enable each one.

| Provider | Env Variable | Notes |
|----------|--------------|-------|
| [Anthropic](https://anthropic.com) | `ANTHROPIC_API_KEY` | **recommended** — Claude models |
| [OpenAI](https://openai.com) | `OPENAI_API_KEY` | GPT-4o, o1 |
| [OpenRouter](https://openrouter.ai) | `OPENROUTER_API_KEY` | 100+ models, one API |
| [Ollama](https://ollama.ai) | — | local, free, no API key |
| [Groq](https://groq.com) | `GROQ_API_KEY` | fast inference |
| [xAI](https://x.ai) | `XAI_API_KEY` | Grok models |
| [DeepSeek](https://deepseek.com) | `DEEPSEEK_API_KEY` | reasoning models |

See the [Model Providers](/model-providers) page for full details.

## Interactive Config

Use the CLI to manage configuration:

```bash
milaidy configure          # interactive config wizard
milaidy config get <key>   # read a config value
milaidy config set <k> <v> # set a config value
```
