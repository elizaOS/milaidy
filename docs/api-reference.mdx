# REST API Reference

The Milady API server runs on port **2138** by default. All endpoints are prefixed with `/api/` unless otherwise noted. Authentication is required when `MILADY_API_TOKEN` is set.

<Warning>
When `MILADY_API_TOKEN` is configured, include the token as a `Bearer` token in the `Authorization` header, or use the pairing flow to obtain it.
</Warning>

## Agent Status

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/status` | Get current agent status, name, model, uptime, autonomy state, and pending restart info |

## Agent Lifecycle

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/agent/start` | Start the agent, enable autonomy |
| `POST` | `/api/agent/stop` | Stop the agent, disable autonomy |
| `POST` | `/api/agent/pause` | Pause the agent (keep uptime, disable autonomy) |
| `POST` | `/api/agent/resume` | Resume a paused agent, re-enable autonomy |
