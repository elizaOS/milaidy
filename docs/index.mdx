**Milady** is a local-first AI assistant built on elizaOS. This documentation is organized for three audiences:

- **Users** who want to install and run Milady reliably.
- **Operators** who need to run, secure, and debug production-like setups.
- **Developers/agents** who want to extend Milady safely with plugins, actions, and integrations.

## What Milady Includes

- **Interfaces**: CLI/TUI, web dashboard, desktop app, mobile app, Chrome extension.
- **Runtime**: elizaOS-based agent runtime with plugin orchestration.
- **Integrations**: model providers, connectors (Discord/Telegram/etc.), and plugin ecosystem.
- **Local state**: configuration/workspace/logs under `~/.milady/`.
